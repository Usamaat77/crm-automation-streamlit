Requirements

  • Excel file (.xlsx or .xls) with invoice data
  • Google Chrome browser
  • CRM credentials (must be already logged in)
  • Invoice data format as specified in the documentation

Getting Started

This tool automates the process of updating invoice records in your CRM system based on data from Excel files.

To use this tool:

  1. Ensure your Excel file contains the necessary columns (Booking No, Supplier, Actual Net Cost)
  2. Make sure you're already logged into your CRM system in a separate Chrome window
  3. Upload your Excel file and follow the steps to configure and run the automation

For detailed instructions, please refer to the README file.
