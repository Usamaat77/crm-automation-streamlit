Automation Process

The automation will perform the following steps for each invoice in your Excel file:

  1. Navigate to the CRM booking list page
  2. Search for the invoice number (with or without "SZ" prefix)
  3. Click on the invoice link to open the details
  4. Click on the "Add Actual Net" button
  5. Set the supplier from your Excel data
  6. Set the actual net cost from your Excel data
  7. Save the changes
  8. Return to the booking list for the next invoice

A log file will be created with details of the automation process and any errors encountered.
